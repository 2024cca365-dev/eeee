
# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Project Outline

*   **index.html**: The main HTML file.
*   **style.css**: Styles for the application.
*   **main.js**: JavaScript logic for generating numbers.

## Current Task: Initial Setup

1.  **Modify `index.html`**: Set up the basic structure for the lottery number generator.
2.  **Modify `style.css`**: Add modern and visually appealing styles.
3.  **Modify `main.js`**: Implement the lottery number generation logic using a Web Component.
